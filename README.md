# Employee Binary Tree

This is a simple **C++ program** that demonstrates how to use a **binary tree** to store and search employee records.

## Features
- Add employee records (ID and name)
- Search employees by ID
- Display the total number of employees in the tree

## How to Compile and Run
Make sure you have a C++ compiler (like `g++`) installed.

```bash
g++ main.cpp -o employee_tree
./employee_tree
